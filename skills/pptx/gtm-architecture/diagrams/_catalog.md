## Shared Constraints (All GM Diagrams)

1. All buyer-facing diagrams should be understandable without narration within 30 seconds.
2. Competitive and feature claims should be sourced or annotated with evidence level.
3. Value claims require at least one quantified data point on the same slide.
4. Color usage follows Cloudwise palette, avoid arbitrary decorative fills.
5. Font sizes: title >= 20pt, body >= 9pt, footnote >= 7pt.
