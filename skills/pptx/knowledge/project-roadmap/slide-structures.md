## Templates

### Template: 标准项目蓝图

- **Source**: manual
- **Uses**: 0
- **Audience**: 管理层 / 技术团队
- **Slide count**: 8
- **Added**: 2026-03-22
- **Last used**: —

**Structure**:
1. Cover — 项目名称 | 蓝图规划 YYYY
2. Project Overview — 项目背景与目标（OKR / 成功标准）
3. Scope & Deliverables — 交付物清单（分组卡片）
4. Timeline — 项目里程碑（甘特式横向时间轴）
5. Team & Resources — 团队结构 / 资源分配（组织图 或 资源表）
6. Risks & Mitigation — 风险矩阵（影响 × 概率二维图）
7. Dependencies — 外部依赖 / 协作方（图标列表）
8. Close — 下一步行动 / 决策请求

**Notes**: 时间轴和甘特图为核心视觉元素；风险矩阵用 2×2 散点或热力表
